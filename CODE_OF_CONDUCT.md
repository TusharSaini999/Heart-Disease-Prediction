# Code of Conduct

## Our Pledge

We, as contributors and maintainers of the **Heart Disease Prediction Web-Based Solution**, pledge to make participation in our community a respectful, inclusive, and harassment-free experience for everyone.

We are committed to creating a welcoming environment for all, regardless of:

- Age
- Gender identity and expression
- Nationality
- Disability
- Religion
- Technical experience
- Personal background

## Responsibilities

Project maintainers are responsible for:

- Clarifying acceptable behavior
- Enforcing this Code of Conduct
- Taking appropriate action when rules are violated
- Protecting community members

Maintainers have the right to remove, edit, or reject contributions that violate this policy.

---

## Scope

This Code of Conduct applies to:

- GitHub repositories
- Pull requests
- Issues
- Discussions
- Community forums
- All official project communication channels

Both online and offline interactions related to this project are covered.

---

## Enforcement

If you experience or witness unacceptable behavior, please report it to the project maintainers.

Reports will be reviewed confidentially and handled respectfully.

Appropriate actions may include:

- Warning
- Temporary ban
- Permanent ban
- Removal from the project

depending on the severity of the violation.
